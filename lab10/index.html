HTML Semantics

What Is It?

What Is It? Why is it Important How is It Used Sources

HTML semantics refers to using tags that clearly describe the meaning and purpose of the content on a webpage. Instead of using generic tags like div and span, semantic HTML uses tags such as header, article, section, and footer to provide better structure and clarity.

The key difference between semantic and non-semantic elements is that semantic elements convey meaning about the content they contain, while non-semantic elements do not provide any information about their content. For example, a header tag indicates that the content inside it is a header, while a div tag does not provide any semantic meaning.

Tags like header define the introductory content of a page, nav is used for navigation links, and main represents the main content of the document. By using these tags make it easier to navigate and understand the structure of a webpage.
