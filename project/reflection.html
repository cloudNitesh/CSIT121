Final Thoughts

Creating this portfolio helped me put together everything I learned this semester. I practiced using structure tags, layout rules, responsive design, tables, lists, and navigation menus. Coding every page by hand helped me understand how important consistency is in web design.

I also learned how media queries improve the way a site looks on phones. Testing my pages on mobile showed me how important spacing, text size, and layout are when the screen becomes smaller. The experience helped me understand responsive design more clearly.

Overall, this project taught me useful skills that I can continue using in future web development work. I now feel confident building simple websites using only HTML and CSS.
